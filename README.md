# Web Application 1 - Assignment Project

### How to Run This Project:
1. **Clone the repository:** Download this code to your local machine.
2. **Update Connection String:** Open `appsettings.json` and ensure the `DefaultConnection` matches your local SQL Server instance.
3. **Apply Migrations:** Open the **Package Manager Console** in Visual Studio and run the command: `update-database`. This creates the Students, Courses, and Instructors tables.
4. **Run the App:** Press **F5**. This will launch the browser and show the **Swagger UI** where you can test the API endpoints.
